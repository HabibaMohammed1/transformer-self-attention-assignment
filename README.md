# 🧠 Transformer Self-Attention Assignment

## 📌 Overview

This project presents a clean implementation of **scaled dot-product self-attention**, a core component of Transformer models.

The implementation is written in Python using NumPy for educational purposes.

---

## ⚙️ Self-Attention Formula

Attention(Q, K, V) = softmax(QKᵀ / √dk) V

---

## 🧩 Key Steps

* Convert input into embeddings
* Compute Query (Q), Key (K), and Value (V)
* Calculate attention scores
* Apply scaling and softmax
* Generate final output
